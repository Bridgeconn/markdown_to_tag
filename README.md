Convert translation notes to markdown. 

Steps to run script: 
Put sorce file on folder with any name.

##Make ruby script executable:
$ sudo chmod +x R markdown_to_tag.rb

##Run script
 $ sudo ./markdown_to_tag.rb

Output file will generate as book_name.
